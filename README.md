# peg-solitaire

Solving the Peg Solitaire puzzle game using backtracking in Kotlin. Read more [here](https://larswaechter.dev/solitaire-backtracking/).

```
 ---------------
|     X X X     |
|     X X X     |
| X X X X X X X |
| X X X o X X X |
| X X X X X X X |
|     X X X     |
|     X X X     |
 ---------------
```
